# 缠论（缠中说禅理论）完整知识体系

## 一、理论基础

缠论由"缠中说禅"在2006-2008年间通过博客系统阐述，是一套**纯粹基于价格走势的几何分析方法**，不依赖任何指标，核心思想是：

> **走势终完美** —— 任何级别的走势都会结束，任何级别的盘整都会结束。

缠论的分析是**递归**的：大级别走势由小级别走势构成，小级别走势又由更小级别构成。

---

## 二、核心概念（由小到大）

### 2.1 K线包含处理

在分析之前，必须先对原始K线进行**包含处理**，消除包含关系：

**包含关系定义**：两根K线，若一根的高低点完全包含另一根（即 H1≥H2 且 L1≤L2），则存在包含关系。

**处理规则**：
- **上升趋势中**（前一根K线重心上移）：取两根K线中**高点的较大值**和**低点的较大值**，合并为一根
- **下降趋势中**（前一根K线重心下移）：取两根K线中**高点的较小值**和**低点的较小值**，合并为一根

**注意事项**：
- 包含处理必须从左到右逐步进行，不能跳跃
- 合并后的K线可能继续与后面的K线存在包含关系，需继续处理
- 趋势方向由包含处理前已确定的方向决定

```
上升包含处理示例：
  原始:  K1(H=10,L=5)  K2(H=12,L=4)   → 包含关系
  合并:  K'(H=12, L=5)   （取高中高、低中高）

下降包含处理示例：
  原始:  K1(H=10,L=5)  K2(H=12,L=4)   → 包含关系
  合并:  K'(H=10, L=4)   （取高中低、低中低）
```

### 2.2 分型

分型是缠论最基本的结构单元，由**经过包含处理后的连续三根K线**构成。

**顶分型**：中间K线的高点是三根中最高的，且中间K线的低点也是三根中最高的
```
条件: H2 > H1 且 H2 > H3 且 L2 > L1 且 L2 > L3

      ╱╲
    ╱╲  ╲╱
  ╱╲  ╲╱    ╱╲
    ╲╱        ╲╱
  K1   K2(顶)  K3
```

**底分型**：中间K线的低点是三根中最低的，且中间K线的高点也是三根中最低的
```
条件: L2 < L1 且 L2 < L3 且 H2 < H1 且 H2 < H3

  ╱╲        ╱╲
    ╲╱  ╱╲╱╲
      ╲╱
  K1   K2(底)  K3
```

**分型的强弱判断**：
- **强顶分型**：第三根K线的收盘价低于第一根K线的低点
- **强底分型**：第三根K线的收盘价高于第一根K线的高点
- 分型的强弱影响后续走势延续的概率

### 2.3 笔

笔是连接**相邻顶分型和底分型**之间的线段，是缠论中最基本的走势单元。

**成笔条件**（严格定义）：
1. 顶分型和底分型必须**交替出现**（不能两个顶分型相连）
2. 顶分型的最高点和底分型的最低点之间，至少有**1根独立K线**（即不属于任何分型的K线）
3. 这意味着一笔至少包含**5根**经过包含处理的K线
4. 顶分型的最高点必须高于底分型的最低点（否则不是有效的笔）

**笔的方向**：
- **向上笔**：从底分型到顶分型，价格上升
- **向下笔**：从顶分型到底分型，价格下降

**笔的修正**：
- 当同类型分型连续出现时，保留更极端的那个（顶分型取更高的，底分型取更低的）
- 新笔可能会破坏已有的笔，需要从左到右重新构建

### 2.4 线段

线段由**至少3笔**组成，是笔的组合。

**线段的方向**：
- **向上线段**：由向上笔开始，整体价格重心上移
- **向下线段**：由向下笔开始，整体价格重心下移

**线段破坏（线段结束条件）**：

线段被破坏分为两种情况：

1. **第一种破坏**（笔破坏）：出现一笔，其方向与线段相反，并且突破了线段起始笔的起点
   - 向上线段中，出现一笔向下跌破线段起始底分型的低点
   - 向下线段中，出现一笔向上突破线段起始顶分型的高点

2. **第二种破坏**（段破坏）：在线段内部，出现与线段方向相反的走势，其特征序列构成对线段方向的否定
   - 需要用到**特征序列**的概念

**特征序列**：
- 向上线段中，所有向下笔构成特征序列
- 向下线段中，所有向上笔构成特征序列
- 特征序列可以像K线一样进行包含处理
- 当特征序列出现分型时，线段可能被破坏

### 2.5 走势类型

走势类型分为**趋势**和**盘整**两种：

- **盘整**：只包含**一个**中枢的走势
- **趋势**：包含**两个或以上**同向中枢的走势
  - 上涨趋势：中枢依次抬高（后一个中枢的区间高于前一个）
  - 下跌趋势：中枢依次降低

### 2.6 走势的递归性

- 1分钟级别的走势构成5分钟级别的笔
- 5分钟级别的笔构成5分钟级别的线段
- 5分钟级别的线段对应15分钟/30分钟级别的走势
- 以此递归上推

**级别对应关系**（参考）：
```
1分钟笔 → 1分钟线段 ≈ 5分钟走势
5分钟笔 → 5分钟线段 ≈ 30分钟走势
30分钟笔 → 30分钟线段 ≈ 日线走势
日线笔 → 日线线段 ≈ 周线走势
```

## 三、中枢

### 3.1 中枢定义

中枢是缠论中最重要的概念，代表**多空力量的平衡区域**。

**严格定义**：某级别走势中，至少**3个连续次级别走势**的价格重叠区间。

**简化定义（笔中枢）**：至少3笔的价格重叠区间。

### 3.2 中枢的四个价格

```
GG ─────────── 中枢最高点（所有笔的最高价）
ZG ─────────── 中枢上沿（各笔高点的最小值）
               中枢区间
ZD ─────────── 中枢下沿（各笔低点的最大值）
DD ─────────── 中枢最低点（所有笔的最低价）
```

- **ZG**（中枢上沿）= min(各笔的高点) —— 重叠区间的上边界
- **ZD**（中枢下沿）= max(各笔的低点) —— 重叠区间的下边界
- **GG**（中枢最高）= max(各笔的高点)
- **DD**（中枢最低）= min(各笔的低点)

**中枢成立条件**：ZG > ZD（必须存在重叠区间）

### 3.3 中枢的扩展与升级

**中枢扩展**：新的笔与中枢有交集（进入ZG-ZD区间），则中枢扩展，笔被纳入中枢。ZG和ZD不变，GG和DD可能更新。

**中枢升级**：当中枢扩展的笔数达到9笔以上，中枢可能升级为更高级别的中枢。

**中枢新生**：新的笔完全离开中枢区间（不与ZG-ZD重叠），形成新的中枢或趋势延续。

### 3.4 中枢的实战意义

- 中枢是多空博弈的战场，价格在中枢内震荡说明多空力量均衡
- 突破中枢上沿（ZG）= 多头占优，可能上涨
- 跌破中枢下沿（ZD）= 空头占优，可能下跌
- 中枢越大（持续时间越长），突破后的力度越大
- 中枢的中心价格 = (ZG + ZD) / 2，常作为支撑/阻力参考

## 四、背驰

背驰是缠论中判断走势转折的核心工具。

### 4.1 背驰的本质

**走势力度递减** —— 价格在同一方向上继续运动，但推动力度在减弱。

### 4.2 背驰的分类

#### 趋势背驰（标准背驰）

发生在趋势（至少两个中枢）的最后阶段：
- **上涨趋势背驰**：最后一段上涨的力度 < 前一段上涨的力度
- **下跌趋势背驰**：最后一段下跌的力度 < 前一段下跌的力度

趋势背驰是最可靠的转折信号，对应**第一类买卖点**。

#### 盘整背驰

发生在盘整（一个中枢）中：
- 离开中枢的走势力度 < 进入中枢前的走势力度
- 盘整背驰可能导致中枢继续震荡，不一定反转

#### 小级别背驰（盘整背驰的变种）

- 在次级别走势中出现的背驰
- 可能仅导致次级别的调整，不影响主级别走势

### 4.3 力度比较方法

**方法一：MACD辅助判断**（最常用）
- 比较两段同向走势对应的MACD柱状图面积（DIF与DEA之间的面积）
- 后一段面积 < 前一段面积 → 背驰
- 黄白线（DIF/DEA）回抽零轴后的高度也可辅助判断

**方法二：价格幅度比较**
- 后一段走势的价格幅度 < 前一段
- 简单但不够精确

**方法三：成交量比较**
- 后一段走势的成交量 < 前一段
- 量价背离也是背驰的表现

**方法四：笔的斜率比较**
- 后一笔的斜率（价格变化/时间）< 前一笔
- 适合程序化判断

### 4.4 背驰判断的注意事项

- 比较的两段走势必须是**同向**的（都是上涨或都是下跌）
- 两段走势之间必须经过**同一中枢**
- MACD的DIF必须**回抽零轴**后再次离开，才能做有效对比
- 背驰只是转折的**必要条件**，不是充分条件

## 五、三类买卖点

### 5.1 第一类买卖点（趋势背驰买卖点）

**第一类买点（一买）**：
- 条件：下跌趋势中，最后一段下跌出现背驰
- 位置：最后一个向下笔的终点（底分型处）
- 置信度：最高（0.85-0.95）
- 风险：如果是盘整背驰而非趋势背驰，可能继续下跌

**第一类卖点（一卖）**：
- 条件：上涨趋势中，最后一段上涨出现背驰
- 位置：最后一个向上笔的终点（顶分型处）
- 置信度：最高（0.85-0.95）

```
一买示意图：
         ╱╲
        ╱  ╲       ╱╲
       ╱    ╲     ╱  ╲
      ╱      ╲   ╱    ╲
     ╱        ╲ ╱      ╲
              中枢       ╲
                          ╲  ← 力度减弱（背驰）
                           ╲
                            ★ 一买点
```

### 5.2 第二类买卖点（回调不破买卖点）

**第二类买点（二买）**：
- 条件：一买出现后，价格上涨一段再回调，回调的低点**不跌破一买的低点**
- 位置：回调的最低点处
- 置信度：较高（0.70-0.80）
- 意义：确认一买有效，趋势反转成立

**第二类卖点（二卖）**：
- 条件：一卖出现后，价格下跌一段再反弹，反弹的高点**不突破一卖的高点**
- 位置：反弹的最高点处

```
二买示意图：
                              ╱╲
                    ╱╲       ╱
           ╱╲     ╱  ╲     ╱
          ╱  ╲   ╱    ╲   ╱
         ╱    ╲ ╱      ╲ ╱
        ╱      ★一买    ★二买
       ╱      （低点）  （更高的低点）
```

### 5.3 第三类买卖点（中枢突破回踩买卖点）

**第三类买点（三买）**：
- 条件：价格向上离开中枢（突破ZG），回踩时低点**不跌回中枢区间**（仍在ZG之上）
- 位置：回踩的最低点处
- 置信度：高（0.75-0.85）
- 意义：确认中枢突破有效，后续可能形成更高级别上涨

**第三类卖点（三卖）**：
- 条件：价格向下离开中枢（跌破ZD），反弹时高点**不回到中枢区间**（仍在ZD之下）
- 位置：反弹的最高点处

```
三买示意图：
                              ╱╲ ← 突破
          ╱╲    ╱╲          ╱
    ZG ──╱──╲──╱──╲────────╱────
        ╱    ╲╱    ╲      ╱
    ZD ╱──────────────╲──╱──────
                       ╲╱
                       ★三买（回踩不入中枢）
```

### 5.4 买卖点的组合使用

- **最安全的买入**：一买 + 二买 + 三买 同时在不同级别出现
- **一买**适合左侧交易者（逆势），风险较高但空间最大
- **二买**是确认信号，适合稳健型
- **三买**适合右侧交易者（顺势），确定性最高但空间较小
- 实战中，**二买和三买**的成功率通常高于一买

## 六、区间套

区间套是缠论中**精确定位买卖点**的方法。

### 6.1 原理

通过从大级别到小级别逐级缩小分析范围，精确定位转折点：

```
日线级别: 确定大方向（下跌趋势背驰区间）
  → 4小时级别: 在日线背驰区间内，找4小时的背驰
    → 1小时级别: 在4小时背驰区间内，找1小时的背驰
      → 15分钟级别: 精确定位买点
```

### 6.2 操作步骤

1. 在大级别确认趋势背驰（一买/一卖条件成立）
2. 切换到次级别，在背驰区间内寻找次级别的走势终点
3. 继续向下切换级别，逐步缩小范围
4. 在最小可操作级别执行买入/卖出

### 6.3 实战价值

- 大大提高入场精度，减少浮亏
- 但需要有足够的看盘时间和操作频率
- 适合短线和波段交易者

## 七、多义性

### 7.1 什么是多义性

同一段走势可能有多种合理的分解方式（笔的划分、中枢的认定等），这是缠论的固有特征。

### 7.2 处理多义性的原则

1. **当下原则**：以最新K线为基准，选择最能反映当前走势状态的分解
2. **分类讨论**：列出所有可能的分解，分析每种情况下的操作策略
3. **级别统一**：同一次分析中，保持级别的一致性
4. **取最严格**：当不确定时，取对自己最不利的分解来制定策略

## 八、实战应用框架

### 8.1 分析步骤

```
1. 确定分析级别（日线、4小时、1小时等）
2. K线包含处理
3. 标记分型（顶分型、底分型）
4. 连接笔（向上笔、向下笔）
5. 构建线段
6. 识别中枢（ZG、ZD、GG、DD）
7. 判断走势类型（趋势 or 盘整）
8. 检查背驰条件
9. 识别买卖点
10. 结合多级别区间套精确入场
```

### 8.2 交易规则

**买入条件**（至少满足一项）：
- 出现一买/二买/三买信号
- 多级别共振（大小级别同时出现买点）
- 背驰确认 + 分型确认

**卖出条件**（至少满足一项）：
- 出现一卖/二卖/三卖信号
- 买入级别的走势完成（走势终完美）
- 止损位被触发

**仓位管理**：
- 一买：轻仓试探（20-30%）
- 二买确认：加仓至中等（50-60%）
- 三买确认：满仓或接近满仓（80-100%）
- 出现卖点信号：逐步减仓

### 8.3 风控要点

- 一买的止损位：低于一买低点的一定比例（如3-5%）
- 二买的止损位：低于一买的低点
- 三买的止损位：跌回中枢区间内（跌破ZG）
- 严格执行止损，不抱有幻想

## 九、程序化实现要点

### 9.1 数据需求

- 至少100根K线才能有效识别缠论结构
- 多时间框架数据用于区间套分析
- 实时K线推送用于动态更新分型和笔

### 9.2 算法复杂度

| 步骤 | 复杂度 | 说明 |
|------|--------|------|
| 包含处理 | O(n) | 单次遍历 |
| 分型识别 | O(n) | 滑动窗口 |
| 笔构建 | O(n) | 线性扫描 |
| 线段构建 | O(n) | 逐步推进 |
| 中枢识别 | O(n) | 滑动计算重叠 |
| 背驰判断 | O(1) | 只比较最后两段 |
| 买卖点 | O(1) | 基于最新笔和中枢 |

### 9.3 背驰量化方法

```
力度 = |结束价 - 起始价| / 笔的K线根数
     = 价格变动幅度 / 时间

背驰条件: 最后一笔力度 < 前同向笔力度
```

更精确的方法：结合MACD柱状图面积
```
MACD面积 = Σ|histogram[i]|  (i从笔起点到终点)
背驰条件: 最后一段MACD面积 < 前同向段MACD面积
```

**方法三：成交量背驰（已实现）**
```
笔均量 = 笔内K线成交量之和 / 笔内K线根数
背驰条件: 最后一笔均量 < 前同向笔均量（量价背离）
```

成交量背驰的意义：
- 价格创新高/新低但成交量萎缩 → 推动力耗尽，趋势即将反转
- 与MACD面积背驰形成双重确认时，信号可靠性显著提升

### 9.4 当前项目中的实现

项目中缠论模块位于：
- 数据模型: `service/indicator/chan/` (ChanFractal, ChanBi, ChanSegment, ChanZhongshu, ChanSignalPoint, ChanResult)
- 核心计算: `service/indicator/ChanCalculator.java`
- 交易策略: `service/strategy/ChanStrategy.java`

计算流程: `K线 → 包含处理 → 分型识别 → 笔构建 → 线段构建 → 中枢识别 → 买卖点判断(四重验证) → Signal`

**四重背驰验证（第一类买卖点）：**
1. 价格幅度背驰：最后一笔幅度 < 前同向笔幅度
2. MACD面积背驰：最后一笔MACD面积 < 前同向笔面积
3. 斜率背驰：最后一笔斜率 < 前同向笔斜率
4. 成交量背驰：最后一笔平均成交量 < 前同向笔平均成交量

前三维至少满足2个即可触发信号，成交量背驰作为独立加分项（+0.03置信度）。

**成交量确认（第三类买卖点）：**
- 突破中枢时放量（>中枢均量120%）→ 突破有效，置信度+0.05
- 回踩时缩量（<突破笔80%）→ 回踩只是洗盘，置信度+0.03
- 突破时缩量（<中枢均量80%）→ 假突破风险，置信度-0.05
